## 원본 안내 및 이 포크 본에 대한 안내

* 이 repo는 Gumball12/lmsplusplus 부가기능을 Firefox용으로 포팅하는 프로젝트입니다.
* 원작자와 합의되지 않았으므로 Firefox Add-Ons에는 업로드하지 않습니다.
* 추후 원작자에게 원본 소스를 전달할 수 있습니다.
* Chrome용으로 재포팅하지 않습니다.

안녕하세요.
